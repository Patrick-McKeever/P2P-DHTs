Detailed Description

remote_peer.h

This file exists to implement several classes:

  • RemotePeer, which a locally-run peer will use to represent remote peers.
  • ThreadSafeRemotePeer, a wrapper which provides thread-safe access and mutation of RemotePeer instances. This class exists to represent peers. Peers will store instances of this class to represent their successors, predecessors, and finger table entries.
