Detailed Description

template<int key_base, int key_len>
class ThreadSafeKey< key_base, key_len >

In most cases, it is not necessary to protect a key with read-write mutex locking, since most keys in our program are static. However, since a few cases exist in which keys will be read and modified by several threads, it is pertinent that we protect these keys under read-write locks. This class simply implements a wrapper of GenericKey with a read-write locked mutator and accessor for this Key instance.
