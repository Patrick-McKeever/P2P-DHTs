Detailed Description

The DataBlock class represents a piece of data corresponding to a key. Its implementation should allow the ability to:

  • Encode into fragments using an information dispersal algorithm.
  • Decode from only a fraction of the total fragments generated for any given block (e.g. 14 fragments generated, only 10 needed to decode.
  • Serialize as a string.
  • Construct from a serialized string (e.g. parse the string).
  • Construct from a vector of DataFragments. This will enable replication of data on multiple peers.
