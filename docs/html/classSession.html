Detailed Description

template<typename ReqHandlerType>
class Session< ReqHandlerType >

"Session" class represents a single connection with a client. Inherits from boost::enable_shared_from_this to allow construction of shared_ptr<Session> inside member functions via CRTP (asio is weird with lifetime management).
