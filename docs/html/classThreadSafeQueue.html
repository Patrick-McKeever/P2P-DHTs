Detailed Description

template<typename T>
class ThreadSafeQueue< T >

thread_safe_queue.h

Numerous classes (most notably the server<T> class) use boost::circular_buffer as a means of creating a fixed-length FIFO queue. This is ideal for purposes of logging. However, since many applications require concurrent access of boost::circular_buffer from multiple threads, it is prudent to create a simple, thread-safe wrapper around boost::circular_buffer. The class ThreadSafeQueue should accomplish this by maintaining a circular buffer as a private member and implementing read-write locking around the most relevant methods of the boost::circular_buffer class. Thread safe wrapper around circular queue.
