Detailed Description

The IDA will produce a 2D matrix of ints. This matrix can be reconstructed in full from only a handful of the rows in that matrix, so long as the index of those rows in the matrix are known. This data structure exists to represent a single row. It should:

  • Hold the vector of doubles corresponding to a single row.
  • Hold the index of said row.
  • Be able to be serialized into a string.
