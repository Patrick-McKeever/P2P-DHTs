Detailed Description

template<int key_base, int key_len>
class GenericKey< key_base, key_len >

Generic class for keys in a logical ring. The behavior of this type depends upon the size of the ring, which will be computed from template parameters.

Template Parameters
key_baseThe base of the key. E.g. 16 for a hex key.
key_lenThe maximum number of digits in the key.
